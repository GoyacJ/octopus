# 财务报表分析参考

## 一、三大报表结构

### 资产负债表（时点报表）
```
资产 = 负债 + 所有者权益
```
- 资产按流动性排列：流动资产 → 非流动资产
- 负债按到期日排列：流动负债 → 非流动负债

### 利润表（期间报表）
```
营业收入 - 营业成本 - 税金及附加 - 期间费用 = 营业利润
营业利润 + 营业外收支 = 利润总额
利润总额 - 所得税费用 = 净利润
```

### 现金流量表（期间报表）
- 经营活动现金流（核心造血能力）
- 投资活动现金流（资本支出与投资回报）
- 筹资活动现金流（融资与分配）

## 二、核心财务指标体系

### 偿债能力
| 指标 | 公式 | 参考值 | 说明 |
|------|------|-------|------|
| 流动比率 | 流动资产 ÷ 流动负债 | 1.5-2.0 | 过低短期偿债风险 |
| 速动比率 | (流动资产-存货) ÷ 流动负债 | ≥1.0 | 排除变现慢的存货 |
| 资产负债率 | 总负债 ÷ 总资产 × 100% | 40%-65% | 超高财务杠杆风险大 |
| 利息保障倍数 | EBIT ÷ 利息费用 | ≥3.0 | 低于1无法覆盖利息 |
| 现金比率 | 货币资金 ÷ 流动负债 | ≥0.2 | 最保守的偿债指标 |

### 盈利能力
| 指标 | 公式 | 说明 |
|------|------|------|
| 毛利率 | (收入-成本) ÷ 收入 × 100% | 产品基础盈利能力 |
| 净利率 | 净利润 ÷ 收入 × 100% | 综合盈利水平 |
| ROA（总资产收益率） | 净利润 ÷ 平均总资产 × 100% | 资产运用效率 |
| ROE（净资产收益率） | 净利润 ÷ 平均净资产 × 100% | 股东回报率（核心指标） |
| ROIC（投入资本回报率） | NOPAT ÷ 投入资本 × 100% | 全部资本运用效率 |

### 营运能力
| 指标 | 公式 | 说明 |
|------|------|------|
| 应收账款周转天数 | 365 ÷ (收入÷平均应收) | 越短回款越快 |
| 存货周转天数 | 365 ÷ (成本÷平均存货) | 越短周转越快 |
| 总资产周转率 | 收入 ÷ 平均总资产 | 资产使用效率 |
| 应付账款周转天数 | 365 ÷ (成本÷平均应付) | 付款周期 |

### 发展能力
| 指标 | 公式 | 说明 |
|------|------|------|
| 收入增长率 | (本期收入-上期) ÷ 上期 × 100% | 增长势头 |
| 净利润增长率 | (本期净利润-上期) ÷ 上期 × 100% | 盈利增长 |
| 总资产增长率 | (本期总资产-上期) ÷ 上期 × 100% | 扩张速度 |

## 三、杜邦分析法

ROE 拆解为三个驱动因素：
```
ROE = 净利率 × 总资产周转率 × 权益乘数
     = (净利润/收入) × (收入/总资产) × (总资产/净资产)
```

| 因素 | 含义 | 提升方向 |
|------|------|---------|
| 净利率 | 盈利能力 | 控成本、提单价、优化产品结构 |
| 总资产周转率 | 营运效率 | 加快周转、处置闲置资产 |
| 权益乘数 | 财务杠杆 | 适度负债（过高则风险大） |

## 四、现金流分析

### 现金流匹配判断
| 状态 | 经营 | 投资 | 筹资 | 判断 |
|------|------|------|------|------|
| 成熟期(+) | + | - | - | 正常，现金自给自足 |
| 成长期(+) | + | - | + | 正常，外部融资支持扩张 |
| 衰退期(-) | - | - | + | 警惕，靠融资维持运营 |
| 虚假繁荣 | - | + | + | 警惕，靠出售资产/融资维持 |

### 自由现金流（FCF）
```
FCF = 经营活动现金流 - 资本性支出
```
- FCF > 0：企业有"余粮"可分配或投资
- FCF < 0：企业需要外部输血

## 五、报表粉饰识别

### 利润表常见手法
1. **提前确认收入**：未满足收入确认条件即入账
2. **推迟确认费用**：应本期确认的费用挂账或资本化
3. **非经常性损益包装**：利用资产处置、政府补助等调节利润
4. **存货跌价准备**：计提不充分或集中释放

### 资产负债表常见手法
1. **虚增资产**：虚构应收账款、存货
2. **隐瞒负债**：表外融资、或有负债不披露
3. **关联交易非关联化**：掩盖利益输送

### 识别信号
- 应收账款增速 >> 收入增速
- 存货周转天数异常上升
- 经营现金流持续低于净利润
- 频繁更换审计师
- 关联交易占比过高
- 期末突击大额交易

## 六、分析报告模板

```
## 财务分析报告

### 一、整体概况
- 报告期/对比期
- 核心财务数据摘要（收入/净利润/总资产/净资产）

### 二、盈利分析
- 收入结构与增长
- 毛利率/净利率变动及原因
- ROE杜邦分解

### 三、偿债分析
- 流动性指标
- 资本结构
- 利息保障能力

### 四、营运分析
- 应收/存货/应付周转趋势
- 资产使用效率

### 五、现金流分析
- 三大现金流结构
- 自由现金流状况

### 六、风险提示
- 主要风险点（高/中/低）

### 七、建议
- 改进建议（按优先级排序）
```
