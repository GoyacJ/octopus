# Contract Source

`contracts/` 保存 Octopus Phase 0 冻结后的机器可读契约源。

当前目录约束：

- `v1/core-objects.json` 冻结核心对象最小字段集合与所属边界上下文。
- `v1/enums.json` 冻结跨平面共享枚举值。
- `v1/events.json` 冻结事件骨架与最小 payload 字段。

这些文件是实现层对齐用的正式输入，不替代 [docs/CONTRACTS.md](/Users/goya/Work/weilaizhihuigu/super-agent/octopus/docs/CONTRACTS.md) 的中文说明文档。
