# 税务分析参考

## 一、增值税（VAT）

### 税率（2024年现行）
| 类型 | 税率 | 适用范围 |
|------|------|---------|
| 基本税率 | 13% | 大部分货物、有形动产租赁 |
| 低税率 | 9% | 交通运输、建筑、不动产租赁、农产品等 |
| 低税率 | 6% | 现代服务（除租赁外）、金融服务、生活服务 |
| 零税率 | 0% | 出口货物、跨境应税行为 |

### 小规模纳税人
- 征收率：3%（一般），1%（阶段性优惠至2027年底）
- 月销售额≤10万（季≤30万）：免征增值税

### 关键计算
```
销项税额 = 不含税销售额 × 税率
进项税额 = 买价 × 税率（凭合法扣税凭证）
应纳税额 = 销项税额 - 进项税额
```

### 不得抵扣的进项税额
- 用于简易计税/免税/集体福利/个人消费
- 非正常损失的购进货物及相关劳务
- 贷款服务、餐饮服务、居民日常服务、娱乐服务

## 二、企业所得税（CIT）

### 基本框架
| 项目 | 内容 |
|------|------|
| 法定税率 | 25% |
| 小型微利企业 | 实际税负5%（应纳税所得额≤300万） |
| 高新技术企业 | 15% |
| 技术先进型服务企业 | 15% |
| 研发费用加计扣除 | 一般100%，集成电路/医药等200% |
| 残疾人工资加计扣除 | 100% |

### 应纳税所得额计算
```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 以前年度亏损
应纳所得税额 = 应纳税所得额 × 适用税率 - 减免税额 - 抵免税额
```

### 关键纳税调整项目
- 业务招待费：按发生额60%扣除，且≤当年销售(营业)收入5‰
- 广告宣传费：≤当年销售(营业)收入15%（超可结转）
- 公益性捐赠：≤年度利润总额12%（超可结转3年）
- 职工福利费/教育经费/工会经费：分别≤14%/8%/2%

## 三、个人所得税（IIT）

### 综合所得（工资/劳务/稿酬/特许权使用费）
```
应纳税所得额 = 年收入额 - 60000 - 专项扣除 - 专项附加扣除 - 依法确定的其他扣除
```

### 税率表（综合所得/经营所得）
| 级数 | 全年应纳税所得额 | 税率 | 速算扣除数 |
|------|----------------|------|-----------|
| 1 | ≤36,000 | 3% | 0 |
| 2 | 36,000-144,000 | 10% | 2,520 |
| 3 | 144,000-300,000 | 20% | 16,920 |
| 4 | 300,000-420,000 | 25% | 31,920 |
| 5 | 420,000-660,000 | 30% | 52,920 |
| 6 | 660,000-960,000 | 35% | 85,920 |
| 7 | >960,000 | 45% | 181,920 |

### 专项附加扣除（2023年标准）
| 项目 | 扣除标准 | 备注 |
|------|---------|------|
| 子女教育 | 2000元/月/孩 | 扩大至3岁-博士 |
| 3岁以下婴幼儿照护 | 2000元/月/孩 | - |
| 赡养老人 | 3000元/月 | 独生子女 |
| 房贷利息 | 1000元/月 | 首套住房贷款 |
| 住房租金 | 800-1500元/月 | 按城市分档 |
| 继续教育 | 400/3600元/月 | 学历/职业资格 |
| 大病医疗 | 据实 | 超15000部分，上限80000 |

### 其他所得
- 财产转让所得：20%
- 财产租赁所得：20%
- 利息股息红利所得：20%
- 偶然所得：20%

## 四、其他税种

### 消费税
| 税目 | 税率形式 | 典型税率 |
|------|---------|---------|
| 烟 | 复合计征 | 甲类卷烟56%+0.003元/支 |
| 酒 | 复合/比例 | 白酒20%+0.5元/500g |
| 高档化妆品 | 比例 | 15% |
| 小汽车 | 比例 | 乘用车1%-40% |

### 房产税
- 从价计征：1.2% × 房产原值 × (1-扣除比例10%-30%)
- 从租计征：12% × 租金收入（个人出租住房4%）

### 土地增值税（四级超率累进）
| 级数 | 增值率 | 税率 | 速算扣除系数 |
|------|-------|------|------------|
| 1 | ≤50% | 30% | 0 |
| 2 | 50%-100% | 40% | 5% |
| 3 | 100%-200% | 50% | 15% |
| 4 | >200% | 60% | 35% |

## 五、税务筹划要点

### 常见合规筹划方向
1. **组织形式选择**：有限责任公司 vs 个人独资/合伙（所得税穿透）
2. **纳税人身份**：一般纳税人 vs 小规模（视业务结构选择）
3. **高新技术企业认定**：享受15%优惠税率
4. **研发费用加计扣除**：规范研发费用归集
5. **区域性优惠**：海南自贸港（15%）、西部大开发（15%）、自贸试验区
6. **递延纳税**：合理利用纳税义务发生时间差异
7. **公益性捐赠**：提前规划在利润总额12%限额内

### ⚠️ 红线提示
- 严禁虚开发票
- 严禁转移隐匿收入
- 关联交易需符合独立交易原则
- 税务筹划必须以真实业务为前提
