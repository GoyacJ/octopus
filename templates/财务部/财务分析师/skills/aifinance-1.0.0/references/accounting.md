# 会计核算参考

## 一、中国企业会计准则体系（CAS）

| 准则编号 | 准则名称 | 核心要点 |
|---------|---------|---------|
| CAS 1 | 存货 | 先进先出/加权平均/个别计价，跌价准备 |
| CAS 4 | 固定资产 | 折旧方法（年限平均/双倍余额递减/年数总和），减值测试 |
| CAS 8 | 资产减值 | 可收回金额=Max(公允价值-处置费用, 未来现金流现值)，减值不可转回 |
| CAS 9 | 职工薪酬 | 短期/离职后/辞退/其他长期职工薪酬，设定受益计划 |
| CAS 11 | 借款费用 | 资本化条件（资产支出+借款费用发生+购建活动开始），暂停/停止资本化 |
| CAS 13 | 或有事项 | 预计负债确认条件（现时义务+很可能+可靠计量），或有负债/资产披露 |
| CAS 14 | 收入 | 五步法模型（识别合同→识别履约义务→确定交易价格→分摊→确认收入） |
| CAS 16 | 政府补助 | 与资产相关/与收益相关，总额法/净额法 |
| CAS 18 | 所得税 | 资产负债表债务法，递延所得税资产/负债 |
| CAS 20 | 企业合并 | 同一控制（账面价值）vs 非同一控制（公允价值），商誉确认 |
| CAS 21 | 租赁 | 承租人单一模型（使用权资产+租赁负债），出租人分类（融资/经营） |
| CAS 22 | 金融工具 | 分类（FVTPL/FVTOCI/AMC），减值（ECL预期信用损失），嵌入衍生工具 |
| CAS 23 | 金融资产转移 | 终止确认判断（过手测试+风险报酬转移），继续涉入 |
| CAS 24 | 套期会计 | 公允价值套期/现金流量套期/境外经营净投资套期 |
| CAS 33 | 合并财务报表 | 控制判断（权力+可变回报+有能力运用权力），合并程序，少数股东权益 |
| CAS 36 | 关联方披露 | 控制/共同控制/重大影响，交易类型与定价政策 |
| CAS 39 | 公允价值计量 | 三个层级输入值，估值技术（市场法/收益法/成本法） |

## 三、常见业务分录模板

### 收入确认（CAS 14 五步法完成时）
```
借：应收账款/银行存款
    贷：主营业务收入
        应交税费—应交增值税（销项税额）
```

### 采购入库
```
借：原材料
    应交税费—应交增值税（进项税额）
    贷：应付账款/银行存款
```

### 工资计提与发放
```
计提：
借：管理费用/销售费用/制造费用
    贷：应付职工薪酬

发放：
借：应付职工薪酬
    贷：银行存款
        应交税费—应交个人所得税
        其他应收款（社保/公积金个人部分）
```

### 固定资产折旧
```
借：管理费用/销售费用/制造费用
    贷：累计折旧
```

### 期末结转
```
收入结转：
借：主营业务收入 / 其他业务收入 / 投资收益 / 营业外收入
    贷：本年利润

成本费用结转：
借：本年利润
    贷：主营业务成本 / 其他业务成本 / 税金及附加
        销售费用 / 管理费用 / 财务费用
        资产减值损失 / 营业外支出 / 所得税费用
```

## 四、长期股权投资

### 核算方法选择
| 持股比例 | 影响程度 | 核算方法 |
|---------|---------|---------|
| <20% | 无重大影响 | 金融资产（FVTPL/FVTOCI） |
| 20%-50% | 重大影响 | 权益法 |
| >50% | 控制 | 成本法 + 合并报表 |

### 权益法核心处理
- 初始投资成本调整：比较初始成本与享有被投资方可辨认净资产公允价值份额
- 确认投资损益：按持股比例调整被投资单位净利润（抵消未实现内部交易损益）
- 其他综合收益/其他权益变动：按比例确认

## 五、合并报表

### 合并范围判断（控制三要素）
1. 对被投资方拥有权力
2. 享有或承担可变回报的风险或权利
3. 有能力运用权力影响回报金额

### 合并抵销分录（主要）
- 母公司长期股权投资 vs 子公司所有者权益
- 内部应收应付
- 内部收入与成本
- 内部存货未实现利润
- 内部固定资产未实现利润
